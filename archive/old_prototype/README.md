# Old Prototype Archive

This directory preserves the earlier CLI-first prototype that used:

- `src/agents/*`
- `src/models.py`
- `src/orchestrator.py`
- `src/report.py`
- legacy model and structured-output tests

It is archived for reference only. The active implementation is the API-first
workflow in `src/workflow.py`, `src/workflow_agents.py`, `src/workflow_models.py`,
and `src/api.py`.
